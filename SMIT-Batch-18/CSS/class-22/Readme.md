## Figma Assignment

For this assignment, you will create a **complete webpage** by looking at the Figma template provided here:  
[Figma Design](https://www.figma.com/design/c0gvhzQAZF8O3jhBrcLHbh/SMIT---Web-Batch-14?node-id=25-145&p=f).  

Your task is to **reproduce the design exactly** as shown in the template using **HTML, CSS, and Bootstrap**, making sure all sections, colors, fonts, and layout match the design.  
At the same time, ensure that the webpage is **fully responsive** and works well on desktop, tablet, and mobile screens.  

Once finished, upload your project to **GitHub** and submit the link for evaluation.
